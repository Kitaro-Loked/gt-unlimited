# Contributor Covenant Code of Conduct / 行为准则

## Our Pledge / 我们的承诺

We as members, contributors, and leaders pledge to make participation in our community a harassment-free experience for everyone, regardless of age, body size, visible or invisible disability, ethnicity, sex characteristics, gender identity and expression, level of experience, education, socio-economic status, nationality, personal appearance, race, religion, or sexual identity and orientation.

我们作为成员、贡献者和领导者，承诺让每个人都能在我们的社区中获得无骚扰的参与体验，无论其年龄、体型、可见或不可见的残疾、种族、性别特征、性别认同与表达、经验水平、教育程度、社会经济地位、国籍、外貌、种族、宗教或性取向。

## Our Standards / 我们的标准

Examples of behavior that contributes to a positive environment:
- 使用友善和包容的语言
- 尊重不同的观点和经验
- 大方接受建设性批评
- 关注社区最有利的事情

Unacceptable behavior:
- 发表性骚扰或贬损性言论
- 恶意攻击、侮辱或贬低他人
- 公开或私下骚扰
- 未经同意发布他人私人信息

## Enforcement / 执行

Project maintainers are responsible for clarifying and enforcing our standards of acceptable behavior. Reports may be sent to the repository owner via GitHub Issues or Discussions.

项目维护者负责澄清和执行可接受行为标准。违规行为可通过 GitHub Issues 或 Discussions 向仓库所有者举报。

## Scope / 范围

This Code of Conduct applies within all project spaces, and also applies when an individual is officially representing the project in public spaces.

本行为准则适用于所有项目空间，以及个人在公共场合正式代表本项目时。
