# GT UNLIMITED — Terminal Financiera

[English](../README.md) | [中文](README.zh-CN.md) | Español

Un terminal financiero autoalojado de una sola página inspirado en Bloomberg / TradingView. Construido con HTML/CSS/JS vanilla para traders que quieren un panel centralizado de mercados globales, derivados, riesgo, eventos macro y noticias.

## Características principales

- Datos de mercado multiactivo: acciones, cripto, FX, commodities, tasas, ETFs
- Integración con TradingView: gráficos, mapas de calor, análisis técnico
- Opciones, superficie de volatilidad, swaps, productos estructurados
- PnL, atribución, beta/alpha, VaR, pruebas de estrés
- Estados financieros (FA), perfiles de empresa (DES), informes de research (RES)
- Globo 2D/3D interactivo, noticias RSS y lectura de noticias en voz alta

## Inicio rápido

```bash
cd trading.2009731.xyz
node api/proxy-server.js &
caddy run
```
